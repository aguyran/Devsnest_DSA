# Reverse an array

Given an array (or string), the task is to reverse the array/string.

&nbsp;

### Example 1:
```
Input: 
    arr[] = {1, 2, 3}
Output:
    arr[] = {3, 2, 1}
```

### Example 2:
```
Input: 
    arr[] = {4, 5, 1, 2}
Output:
    arr[] = {2, 1, 5, 4}
```

&nbsp;

#### Constraints:
```
1 ≤ N ≤ 1000000
1 ≤ A[i] ≤100000
```
